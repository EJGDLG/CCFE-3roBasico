# Proyecto comentado para alumnos: sitio web de negocio ficticio

Este proyecto es una página web estática en HTML, CSS y Bootstrap. No necesita base de datos ni servidor especial.

## Qué debe modificar el alumno

1. Cambiar el nombre del negocio donde aparece `GYM LA ROCA`.
2. Cambiar los textos de servicios, horarios, lugares y descripción del negocio.
3. Cambiar imágenes dentro de la carpeta `Recursos/`.
4. Revisar los enlaces del menú para que funcionen.
5. Personalizar colores y tamaños en `carousel.css`.

## Archivos principales

- `index.html`: página principal.
- `pag1/index.html`: página de lugares, productos o sucursales.
- `pag2/index.html`: página de servicios o catálogo.
- `pag3/index.html`: página de horarios, contacto o información final.
- `carousel.css`: estilos personalizados.
- `Recursos/`: carpeta donde están las imágenes, Bootstrap, CSS y JavaScript.

## Publicar en GitHub Pages

Para publicarlo fácil, sube el contenido de la carpeta `Web5BACOA07` al repositorio. El archivo `index.html` debe quedar en la raíz del repositorio.

Luego entra a GitHub:

1. Settings.
2. Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /root.
6. Save.

La página se abrirá con una dirección similar a:

`https://usuario.github.io/nombre-del-repositorio/`
